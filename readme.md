# gokuro_c

gokuro_c is a C inplementation of gokuro language.

gokuro language is a simple macro language used to write articles and books of [https://inzkyk.xyz/](https://inzkyk.xyz/).

## building

```
$ gcc main.c
$ clang main.c
$ cl main.c
```

## usage

```
$ gokuro_c < YOUR_FILE
```

## testing

(windows only)

```
$ .\build.bat diff
```
